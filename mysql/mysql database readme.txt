To import a MySQL Database:

1. Go to shell

2. Change directory to: 
/Applications/XAMPP/xamppfiles/bin/mysql

3. Download dump.sql (database copy file) into this directory

4. Enter the following command:
mysql -u [username] -p reference_checker < dump.sql


ALTERNATIVELY,

If you have access to PHPMYADMIN

Click on IMPORT and select the dump.sql file.
